# Backend

Submission to backend assignment

#### 1. Install Dependencies
```js
npm i dependencies
```

Installs all dependencies

#### 2. Database Set Up

Run all commands in `src/db.sql` locally on mysql

#### 3. .env Set up

Define a file: `.env` in src with the format

```env
APP_PORT=3000

JWT_SECRET=<secret>

SALT_ROUNDS=<rounds>

DB_HOST=localhost
DB_USER=<user>
DB_PASSWORD=<password>
DB_NAME=backend
```

#### 4. Run `src/app.js`

#### 5. Enjoy :D
